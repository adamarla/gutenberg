\begin{snippet}
    \correct
    
    If $f'(x) = e^{-x}$, then $f(x)$ is a strictly increasing function 
    
    \reason
    
    $f'(x) = e^{-x} > 0$ for all values of $x$\newline 
    
    It can also never be equal to $0$ \newline 
    
    Hence, $f(x)$ is not just increasing, it is strictly increasing 
    
\end{snippet}
