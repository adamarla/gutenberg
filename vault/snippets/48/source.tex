\begin{snippet}
    
    \incorrect
    
    If $f'(x) = \log x$, then the function $f(x)$ is increasing in $(0,1)$ 
    
    \reason
    
    $\log x < 0$ for $x \in (0,1)$. Hence, $f(x)$ is decreasing -- not increasing -- for $x\in (0,1)$ 
    
\end{snippet}
