\begin{skill}
    \begin{narrow}
         \textcolor{blue}{Three Set Results}
         
         Standard results involving three sets $A,B$ and $C$ 
    \end{narrow}
    
    \reason 
    
The following results can be proven graphically by representing three sets $A,B$ and $C$ in a Venn diagram\newline 

Your textbook will have those proofs. So, we will just list these 
important results 
    \begin{center}
  \begin{tabular}{NN}
   \toprule
        \text{Union} & \left(A\cup B \right) \cup C = A\cup \left(B\cup C \right)\\
   \midrule 
   \text{Intersection} & \left(A\cap B \right) \cap C = A\cap \left(B\cap C \right)\\
    \midrule 
    \text{Mixed} & A\cap \left(B\cup C \right) = \left(A\cap B \right)\cup \left(A\cap C \right) \\
    \midrule 
    & A\cup \left(B\cap C \right) = \left(A\cup B \right)\cap \left(A\cup C \right) \\
    \bottomrule
  \end{tabular}
\end{center}
