\textcolor{blue}{Standard Integrals \#1}

Easiest to remember. Mostly of trigonometric functions. 
\newcard

The table below shows some common derivatives and their \underline{corresponding integrals}

\begin{center}
\begin{tabular}{MM}
\toprule 
\text{Derivative} & \text{Integral} \\ 
\midrule 
\frac{d}{dx}\frac{x^{n+1}}{n+1} = x^n & \int x^n dx= \frac{x^{n+1}}{n+1}+C \\
\midrule 
\frac{d}{dx}x = 1 & \int dx = x + C \\
\midrule 
\dfrac{d}{dx}\sin x = \cos x & \int\cos x\cdot dx = \sin x + C \\
\midrule 
\frac{d}{dx}(-\cos x ) = \sin x & \int\sin x \cdot dx = -\cos x + C \\
\midrule 
\frac{d}{dx}\tan x = \sec^2 x & \int\sec^2x \cdot dx = \tan x + C \\
\midrule 
\frac{d}{dx}\cot x = -\csc^2 x & \int\csc^2 x\cdot dx = -\cot x + C \\
\midrule 
\frac{d}{dx}\sec x = \sec x\tan x & \int\sec x\tan x dx = \sec x + C \\
\midrule 
\frac{d}{dx}\csc x = -\csc x \cot x & \int \csc x\cot x dx = -\csc x + C \\
\midrule
\frac{d}{dx}e^x = e^x & \int e^x \cdot dx = e^x + C \\
\midrule 
\frac{d}{dx}\log \vert x \vert = \frac{1}{x} & \int \frac{dx}{x} = \log\vert x\vert + C \\
\bottomrule
\end{tabular} 
\end{center}
